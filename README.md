# 📦 Applications Included

## 1️⃣ Quote Generator App

### Description
The **Quote Generator App** generates random inspirational quotes for users.  
It is designed to provide motivation, positivity, and thoughtful messages through a clean and simple user interface.

### Features
- Generates random quotes
- Simple and user-friendly UI
- Refresh button to generate new quotes
- Lightweight and fast

### Use Case
Helpful for users who want **daily motivation or inspirational thoughts**.

---

## 2️⃣ Study Quiz Cards App

### Description
The **Study Quiz Cards App** helps students prepare for exams using interactive quiz cards.  
Students can review questions and select answers, while teachers can add new study cards for upcoming tests and revision.

### Features
- Interactive quiz cards
- Multiple choice questions
- Answer selection for practice
- Teachers can create new cards for future tests
- Useful for revision and self-assessment

### Use Case
Designed for **students and teachers** to make studying more interactive and effective.
